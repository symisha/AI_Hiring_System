# 🚀 AI-Powered Automated Hiring System

[![License](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE) [![Python](https://img.shields.io/badge/Python-3.11-blue)](https://www.python.org/) [![React](https://img.shields.io/badge/React-18-blue)](https://reactjs.org/)

## 🎯 Overview

The **AI-Powered Automated Hiring System** is a cutting-edge recruitment solution that automates candidate evaluation through **AI-driven resume screening and intelligent interviews**. It ensures **fair, efficient, and role-specific hiring**, providing detailed insights for recruiters while minimizing human bias.

---

## ⚡ Features

### 🔹 AI Resume Screening

* Automated evaluation of resumes against job requirements.
* Candidate ranking and top picks for recruiter review.
* Reduces bias through structured, data-driven assessment.

### 🔹 AI Interview System

* Conducts automated interviews via AI conversational agents.
* Analyzes candidate responses for skills, communication, and experience.
* Generates detailed interview summaries and scoring.

### 🔹 Employer Portal

* Tailors hiring workflow based on detailed employer questionnaires.
* Dashboard provides job stats, candidate metrics, and AI recommendations.

### 🔹 Detailed Hiring Reports

* Generates PDF reports with candidate info, rejection reasons, and top picks.
* Supports final decision-making with actionable insights.

### 🔹 Bias Reduction

* Uses prompt engineering and diverse datasets to ensure fair evaluation.

---

## 🛠️ Tech Stack

* **Frontend:** React.js + Tailwind CSS
* **Backend:** FastAPI
* **Database:** Supabase
* **AI/ML:** OpenAI GPT API
* **Reporting:** PDF generation libraries

---

## 🏗️ Architecture

1. Employer registers and submits job requirements.
2. Candidates apply online with resumes.
3. AI screens resumes and shortlists top candidates.
4. AI conducts automated interviews.
5. Generates a detailed report with recommendations.
6. Human recruiter makes final hiring decisions.

---

## 💡 Benefits

* Saves time by automating resume screening and interviews.
* Objective evaluations reduce human bias.
* Customizable per role and organization.
* Scalable for multiple job postings and candidate pools.
